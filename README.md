# MongoDB and Elasticsearch on Docker

MongoDBに全文検索エンジンを導入しようと思い構築したDocker環境です。

後日、イメージサイズを抑える工夫や、k8sに対応したりする予定です。

## Environment

ホストOSは、Linuxを想定しています。

Docker（docker-compose）のインストール済みが前提条件です。

MongoDB Compassがあると便利です。
